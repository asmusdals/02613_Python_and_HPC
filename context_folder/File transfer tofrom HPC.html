File Transfer to/from HPC

This document details some different ways to transfer files to/from the DTU HPC cluster. We strongly recommend you use the terminal solutions - it is a great skill to have! However, if you strongly prefer a GUI, we also include a guide for FileZilla.

SCP (Terminal)

Good for a quick single-file transfer.

NOTE: If you are not on DTU wifi you may need to connect with a VPN first or set up ssh keys (see: https://www.hpc.dtu.dk/?page_id=4317).

NOTE: SCP should be run from your own computer, i.e., you should not be ssh'ed into the HPC.

Transfer: Personal Computer to HPC

To transfer a file from your personal computer to the HPC, run the following in a terminal on your computer (not while ssh'ed into the HPC!):

scp path/to/file.py <username>@login.hpc.dtu.dk:path/to/file.py

Here, <username> is your DTU user name, e.g., your study number. Type in your DTU password and the transfer will begin. The path on HPC is from your zhome folder. 

Example: Transferring a file from the current folder to a folder on HPC:

scp compute_from_laptop.py <username>@login.hpc.dtu.dk:Documents/02613_PyHPC/compute_from_laptop.py

Transfer: HPC to Personal Computer

To transfer a file from the HPC to your personal computer run the following in a terminal on your computer (not while ssh'ed into the HPC):

scp <username>@login.hpc.dtu.dk:path/to/file.py path/to/file.py

Here, <username> is your DTU user name, e.g., your study number. Type in your DTU password and the transfer will begin. The path on HPC is from your zhome folder. 

Example: Transferring a file on HPC to your current folder:

scp <username>@login.hpc.dtu.dk:Documents/02613_PyHPC/compute_from_hpc.py compute_from_hpc.py

SFTP (Terminal)

More interactive and good for transferring many files.

NOTE: If you are not on DTU wifi you may need to connect with a VPN first or set up ssh keys (see: https://www.hpc.dtu.dk/?page_id=4317).

Connect

To connect to the transfer server, run the following in a terminal on your computer:

sftp <username>@transfer.gbar.dtu.dk

Here, <username> is your DTU user name, e.g., your study number. Type in your DTU password and you should connect. You should then see a prompt like:

Connected to transfer.gbar.dtu.dk.
sftp>

You can navigate around on the HPC with cd and list a directory with ls like a normal ssh terminal. You can even make new directories with mkdir.

Similarly, you can navigate around on your personal laptop with lcd, list directories with lls, and more. Notice, the commands are the same, just with an l (lowercase L for 'local') in front.

A full list of commands is available in man sftp or online here https://man7.org/linux/man-pages/man1/sftp.1.html.

Transfer

Once you've navigated to the correct folders, you can transfer a file from your personal computer to the HPC with:

put file_on_personal_computer.py

You can transfer a file from the HPC to your personal computer with

get file_on_hpc.py

SFTP also supports wildcards, e.g., you download all files in a directory with:

get *

FileZilla (GUI)

SFTP with a GUI for Windows, Mac OSX and Linux.

Connect:

  1. Download and install FileZilla (https://filezilla-project.org/download.php?type=client).
  2. Launch FileZilla
  3. At the top fill in the following fields
    1. Host: login.hpc.dtu.dk
    2. Username: Your username (e.g., DTU study number)
    3. Password: Your DTU password
    4. Port: 22
  4. Press on the button "Quickconnect". Note: if you are not on DTU wifi, you may need to connect with a VPN before using FileZilla.
  5. Say that you accept all risks, and go on (save password as well if you wish).

After this, the top should say something like:

sftp://yourUser@transfer.gbar.dtu.dk

Transfer:

The left panel shows the "Local site" i.e. the folder on your computer. The right panel shows the folder on the "Remote site", i.e. your zhome on HPC. You can navigate in both like a normal file browser.
